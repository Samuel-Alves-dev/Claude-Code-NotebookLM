# Miniguia de Estudos sobre Claude Code com NotebookLM

## Sobre o Projeto

Este projeto foi desenvolvido utilizando o NotebookLM como ferramenta de aprendizagem ativa para estudar o Claude Code, uma solução de programação assistida por inteligência artificial criada pela Anthropic.

O objetivo foi reunir fontes confiáveis, realizar pesquisas direcionadas através de prompts estratégicos e organizar o conhecimento adquirido em um material estruturado para consulta futura.

Durante o estudo foram explorados temas como funcionamento do Claude Code, conhecimentos necessários para sua utilização, erros comuns de iniciantes, aplicação no desenvolvimento de SaaS e sua relação com frameworks de automação.

## Objetivos

- Compreender o funcionamento do Claude Code.
- Identificar os conhecimentos necessários para utilizar a ferramenta de forma eficiente.
- Conhecer os erros mais comuns cometidos por iniciantes.
- Entender como o Claude Code pode acelerar o desenvolvimento de software.
- Avaliar suas vantagens e limitações em comparação com ferramentas tradicionais de automação.
- Praticar o uso do NotebookLM como ferramenta de organização e aprendizado.
